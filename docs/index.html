Access Guide

Start with the access you already have.

Some collaborators can open the live AXP app. Others only have Google Sheets and OSF download access. This guide shows what each group can do now and when someone with app-level access needs to take over.

Use the live app

Open the running questionnaire, submit a response, check the feedback view, and verify how a change behaves in the app.

Work in Google Sheets

Review wording, translations, options, and version tabs in the shared questionnaire sheet.

Use OSF downloads

Find the latest files, understand which download you need, and use codebooks and changelogs without app access.

Use the visual guide

Open the diagrams when you want a fast picture of how sheet edits, app checks, and downloads fit together.

Access types

  • App access: open the live questionnaire, submit a response, use the feedback chart, and verify the running experience.
  • Google Sheets access: review or edit questionnaire content in the shared sheet, including wording, translations, options, and version tabs.
  • OSF access: download the latest public or private files, plus codebooks and changelogs.
  • Some collaborators have all three. Others only have Google Sheets and OSF access.

App access required

Use someone with app-level access whenever the task depends on the live questionnaire, a real submit flow, the feedback screen, or a runtime check after a sheet change.

Common tasks

I need to test the live flow

App access is required. Use the live app page for the verification checklist.

I need to change wording or translations

Google Sheets access is usually enough for the edit itself. Ask an app user to confirm the final live result if behavior or presentation matters.

I need the latest files

OSF access is enough. Download the current bundle, then use the codebook and changelog to interpret it.

I only have Sheets and OSF access

You can still edit the questionnaire and use the downloads. Hand off any live runtime check to someone who can open the app.

Fastest Path

Use the site in this order

  1. 1
    Identify your access level.

    Decide whether you can open the app, only edit the sheet, only download from OSF, or some combination of those.

  2. 2
    Open the matching page.

    Use App Access for live checks, Google Sheets for questionnaire changes, and OSF Downloads for data files and supporting documentation.

  3. 3
    Hand off app-only checks when needed.

    If your task depends on the running app and you cannot open it yourself, pass that step to someone with app-level access.
