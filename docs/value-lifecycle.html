Live Survey Access

Open the running survey and use the live experience.

This page is for collaborators who can open the running AXP web survey. Here, live survey access means access to the public or deployed survey page, not code or repo access.

Google Sheets workflow Data examples Diagrams

What live survey access lets you do

  • Open the current live questionnaire.
  • Choose a language and walk the actual running flow.
  • Submit a response and reach the feedback view.
  • Check how wording, options, and required behavior appear in the running survey.
  • Use the feedback comparison controls after submission.
Scope of this page

This page is only about the running survey page and feedback view. Sheet structure and data-file shape are covered on the Google Sheets and Data Examples pages.

Live survey workflow

  1. 1
    Open the app URL you were given.

    Start in the running survey rather than a local or static copy.

  2. 2
    Choose your language.

    Check the live wording in the language that matters for the current task.

  3. 3
    Complete one experience path.

    Move through the questionnaire as a real user would.

  4. 4
    Submit at the final reveal step.

    Do not stop before submit if the task depends on feedback or downstream behavior.

  5. 5
    Review the feedback screen.

    Check the chart, comparison controls, and any other live result behavior that matters for the task.

What reload does

  • Opening or refreshing the survey page in the browser starts a new survey session from the beginning.
  • The app does not currently restore in-progress answers after a full browser refresh.
  • The in-app questionnaire reload does not restart the whole page, but it also is not a full draft-restore feature.
  • If item ids and question structure stay the same, some current in-session values may still remain visible after an in-app questionnaire reload.
  • If the questionnaire structure changes, affected fields can re-render or reset, so questionnaire reload is safest before someone starts answering.
